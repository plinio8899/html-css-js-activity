# Actividad HTML y CSS
Trabajo práctico de maquetación web

## 📋 Descripción
Proyecto de práctica de HTML y CSS que contiene diferentes ejemplos de layouts y diseño web.

## 📄 Páginas disponibles
| Archivo | Descripción |
| ------- | ----------- |
| `index.html` | Página principal con layout normal, header superior y 3 cards de personajes |
| `sidebar.html` | Versión con barra lateral de navegación fija |
| `flexbox.html` | Demostración de todas las propiedades de CSS Flexbox |
| `grid.html` | Demostración de todas las propiedades de CSS Grid Layout |

## ✅ Características implementadas
- Diseño responsivo adaptable a todos los tamaños de pantalla
- Animaciones y efectos hover
- Separación completa entre HTML y CSS
- Tipografías modernas
- Cards con imágenes y botones
- Footer con información de contacto
- Navegación entre todas las páginas

## 🛠️ Tecnologías utilizadas
- HTML5 Semántico
- CSS3
- Flexbox
- CSS Grid
- Google Fonts

## 🚀 Como ejecutar
1. Clona o descarga el proyecto
2. Abre cualquiera de los archivos `.html` en tu navegador web
3. Navega entre las diferentes versiones desde los menús

## 📁 Estructura del proyecto
```
.
├── index.html          # Página principal
├── style.css           # Estilos página principal
├── sidebar.html        # Versión con barra lateral
├── sidebar.css         # Estilos sidebar
├── flexbox.html        # Guía Flexbox
├── flexbox.css         # Estilos Flexbox
├── grid.html           # Guía CSS Grid
├── grid.css            # Estilos CSS Grid
└── README.md           # Este archivo
```
